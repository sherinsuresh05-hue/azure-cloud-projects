# Azure Cloud Projects

This repository contains 3 beginner-level Azure cloud projects.



## Project 1: Virtual Machine (VM)

- Created a Linux Virtual Machine in Microsoft Azure  
- Used B1s free tier configuration  
- Configured SSH access for remote login  



## Project 2: Azure Storage Website

- Created Azure Storage Account  
- Enabled Static Website Hosting  
- Uploaded HTML file (index.html)  
- Hosted website successfully  



## Project 3: Azure SQL Database

- Created Azure SQL Database  
- Created table "Students"  
- Inserted sample data using SQL queries  
- Queried data using Azure Query Editor
